YouTube Engagement ML

Dashboard prediksi engagement video dengan kombinasi metadata pra-tayang, metrik pasca-tayang, dan analisis alignment model.

Unsupervised: MiniBatchKMeans F1=0.5577 Supervised: Soft Voting Ensemble Acc=64.48% Bebas Data Leakage
3 Kelas engagement: low, medium, high
2x Pipeline supervised dan unsupervised dalam satu workflow
API Terhubung ke endpoint FastAPI untuk prediksi real-time
Creator Decision Engine

Bukan sekadar prediksi. Sistem ini membandingkan rencana konten dengan realitas engagement.

Machine learning di proyek ini dipakai sebagai alat bantu keputusan untuk kreator. Metadata video dianalisis sebelum tayang, performa aktual dipetakan setelah tayang, lalu keduanya dicocokkan untuk melihat apakah sinyal awal benar-benar selaras dengan respons audiens.

Sebelum tayang Estimasi peluang Low, Medium, atau High dari metadata.
Setelah tayang Klasterisasi performa aktual dari views, likes, comments.
Validasi Alignment Rate mengukur ramalan vs kenyataan.

Analogi singkat

Unsupervised seperti penyortir nilai setelah ujian selesai. Supervised seperti psikolog yang menebak hasil siswa sebelum ujian, hanya dari biodata awal.

Manfaat praktis

Kreator bisa menguji draf judul dan waktu unggah sebelum publikasi, lalu merevisi metadata jika potensi engagement terlihat rendah.

        Arsitektur Model
      


      

        

          
🔵 Unsupervised — MiniBatchKMeans (Model Terbaik dari 16 Eksperimen)

          

            Pipeline: PowerTransformer(Yeo-Johnson) → StandardScaler → IsolationForest(contamination=3%) → MiniBatchKMeans(k=3, batch=4096)

            Fitur input: views, likes, comments, engagement_rate

            Evaluasi: F1 Weighted Test = 0.5577 | Silhouette = 0.2864
          

        


        

          
🔴 Supervised — Weighted Soft Voting Ensemble

          

            Komponen: ExtraTrees(bobot 2x) + HGB-balanced(1x) + HGB-accuracy(1x) + SGD-TF-IDF(bobot 2x)

            Fitur: title_text (TF-IDF + SVD50) + fitur numerik dari judul + tanggal publish + target encoding channel/country

            Evaluasi: Accuracy = 64.48% | F1 Weighted = 64.06%

            Split: 1:1 Train/Test + undersampling 1:1:1 pada train
